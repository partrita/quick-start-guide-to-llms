대규모 언어 모델(LLM) 입문 가이드

ChatGPT 및 주요 LLM 활용을 위한 전략과 모범 사례

Author

Sinan Ozdemir

Published

February 27, 2026

머리말

“대규모 언어 모델(LLM) 입문 가이드 - 제2판” 에 오신 것을 환영합니다!

이 책은 트랜스포머 모델과 대규모 언어 모델(LLM)을 활용한 다양한 애플리케이션 개발과 고급 기법을 실무적인 관점에서 다룹니다. OpenAI의 최신 모델부터 오픈소스 대안 모델까지, 이론보다는 실습 위주의 코드를 통해 LLM의 생태계를 파악하고 직접 구축해 볼 수 있도록 구성되었습니다.

이 책의 구성

본 책은 세 개의 주요 파트로 나뉘어 있으며, 각 파트별 실습 노트북을 통해 단계별로 학습할 수 있습니다.

제1부 - 대규모 언어 모델 입문

LLM의 기본 개념과 기반 기술을 학습합니다. - 시맨틱 검색: 벡터 임베딩을 활용한 의미 기반 검색 구현. - 프롬프트 엔지니어링: LLM의 성능을 끌어올리기 위한 효과적인 프롬프트 작성법. - RAG 파이프라인: 외부 데이터를 결합하여 정확도를 높이는 검색 증강 생성 시스템. - AI 에이전트: 도구와 협력하여 복잡한 작업을 수행하는 자율 에이전트 구축.

제2부 - LLM의 잠재력 극대화하기

모델을 최적화하고 특정 작업에 맞춰 조정하는 방법을 다룹니다. - 미세 조정(Fine-tuning): BERT 및 OpenAI 모델을 특정 도메인의 데이터로 학습시키기. - 고급 프롬프트 기술: DeepSeek, Llama 4 등 최신 모델을 활용한 고난도 추론 및 출력 검증. - 추천 엔진: 사용자 맞춤형 임베딩 모델 구축.

제3부 - 고급 LLM 활용 기법

실무 배포와 전문화된 시스템 구축을 위한 고급 주제를 탐구합니다. - 멀티모달 시스템: 이미지와 텍스트를 동시에 다루는 VQA(Visual Question Answering) 구축. - 정렬(Alignment): SAWYER 봇 프로젝트를 통한 RLHF(인간 피드백 기반 강화학습) 실습. - 최적화 및 배포: 모델 증류(Distillation), 양자화(Quantization)를 통한 효율적인 운영. - 평가 및 검정: 생성 결과물의 신뢰성을 판단하기 위한 정량적 평가 기법.

실무에 바로 적용 가능한 코드와 함께 LLM의 세계로 뛰어들어 보세요!
